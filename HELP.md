## NAME ##
app - short description

## SYNOPSIS ##
app [OPTIONS]

## OPTIONS ##
        
    -b, --build-no
        just print the build number

    -h, --help
        just show this text
        
    -v NUM, --verbosity NUM
        choose how much info to output: (each level includes those above it. interactive output is always visible)
        -2: fatal errors
        -1: errors
        0: warnings (default)
        1: verbose
        2: debug
